s390base

Helper functions for s390 console tests

copy_testsuite

 copy_testsuite($tc);

Fetch the testcase and common libraries from openQA server. The parameter $tc is the name of testcase and common library. It is the part of script path '$script_path'.

execute_script

 execute_script($script, $scriptargs, $timeout);

Execute $script with arguments $scriptargs and upload STDERR and STDOUT as script logs. The maximum execution time of the script is defined by $timeout.
